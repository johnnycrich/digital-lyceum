Packagecom.oosterwijk.irc
Classpublic class FlexIrcClient
InheritanceFlexIrcClient AsIrcClient
Implementsflash.events.IEventDispatcher

This class implements a Flex-based IRC Client. This class Translates the 'OnIrcEvent() methods from the AsIrcClient Class into Flex Events. The evens are divided into three classes. ServerEvent,UserEvent and ChannelEvent. Each event will be set with a type indicating the exact nature of the irc event that occurred The events will also have properties set relevant to that particular operation. for a list of all the properties that are set for each action, look at the 'OnIrcEvent() methods below.
